## Projects

### Snake Game

A classic snake game built with HTML5, CSS3, and JavaScript.

**Features:**
- Responsive controls using arrow keys
- Score tracking with local high score storage
- Smooth animations and modern UI design
- Progressive difficulty as the snake grows

**How to Play:**
1. Open `snake.html` in your web browser
2. Use arrow keys (↑ ↓ ← →) to control the snake
3. Eat the red food to grow and gain points
4. Avoid hitting walls or yourself
5. Try to beat your high score!

[Play the game](snake.html)
